# Project Identity

## Canonical Path

`C:\GITHUB BUILDS\PaliaHotpotReminder`

## Identity

* Installed-first external Windows desktop helper app.
* Passive only.
* Reads the visible Palia clock region via screen capture and OCR.
* Not a real Palia mod.

## Safety Boundaries

* Do not touch Palia game files.
* Do not read process memory.
* Do not inspect packets or network traffic.
* Do not inject DLLs.
* Do not automate input.
* Do not click, press keys, move the mouse, or control gameplay.
* Do not build an in-game overlay.

## Folder Name Clarification

The `Mods` folder is an organizational label only and does not imply game modification.

## Installer Target

`C:\Tools\PaliaHotpotReminder`

## Distribution

* Installer-first.
* Primary release artifact: `PaliaHotpotReminder-Setup-v3.1.4.exe`.
* Matching checksum artifact: `PaliaHotpotReminder-Setup-v3.1.4.exe.sha256`.
* No portable ZIP release path.

## Current UI Direction

* Fixed-size modular CustomTkinter runtime shell.
* High-Contrast Black + Purple visual style.
* Large-display-aware custom popup board.
* OCR/reminder/process-watcher behavior remains unchanged.
