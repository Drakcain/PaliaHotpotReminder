PaliaHotpotReminder v2.9

Quick Start
1. Install with PaliaHotpotReminder-Setup-v2.9.exe.
2. Launch Palia Hotpot Reminder from the Start Menu.
3. Open Palia.
4. Click Setup Clock once.
5. Click Start Reminder.

Installed Path
- C:\Tools\PaliaHotpotReminder\Hotpot-Remind.exe

Tray
- The app can live in the tray.
- Use tray Exit HPR to fully close it.

Support
- Use Debug / Support if you need a support bundle or OCR checks.
- Logs/debug/exports live under C:\Tools\PaliaHotpotReminder.

Important
- Keep the installed folders beside the EXE.
- HPR does not modify Palia, read memory, inject, hook, inspect network traffic, or automate gameplay.
- HPR only OCRs the clock region you choose.
