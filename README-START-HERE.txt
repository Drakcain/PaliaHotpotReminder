PaliaHotpotReminder v2.8

Quick Start
1. Extract the whole ZIP.
2. Run Hotpot-Remind.exe.
3. Open Palia.
4. Click Setup Clock once.
5. Click Start Reminder.

Tray
- The app can live in the tray.
- Use tray Exit HPR to fully close it.

Support
- Use Debug / Support if you need a support bundle or OCR checks.

Important
- Do not run the EXE from inside the ZIP.
- Keep the folders beside the EXE after extraction.
- HPR does not modify Palia, read memory, inject, hook, or automate gameplay.
- HPR only OCRs the clock region you choose.
